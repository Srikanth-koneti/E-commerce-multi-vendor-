## 📌 Project Overview

**MarketHub** is a fully responsive, feature-rich multi-vendor e-commerce marketplace built with pure HTML, CSS, and JavaScript (no frameworks). It simulates a real-world platform like Amazon or Flipkart where multiple vendors can list products under one roof.

---

## ✨ Key Features

### 🏬 Multi-Vendor System
- 6 verified vendor profiles with unique categories and ratings
- Vendor cards with product counts, rating, and badges
- Click vendor to filter their category products

### 🛍️ Product Catalog (16 Products)
- 5 categories: Electronics, Fashion, Home & Kitchen, Books, Sports
- Product cards with emoji icons, ratings, discount badges
- Grid and List view toggle

### 🔍 Search & Filter
- Real-time search by product name, vendor, or category
- Category filter via header dropdown and nav links
- Price range slider (up to ₹5000)
- Sort by: Featured / Price Low-High / Price High-Low / Top Rated / Newest

### 🛒 Shopping Cart
- Add / remove items
- Quantity adjustment (+/−)
- Auto-calculates subtotal, delivery charges, and total
- Free delivery on orders above ₹499
- One-click checkout with confirmation

### ❤️ Wishlist
- Toggle wishlist per product
- Persisted in localStorage
- Count badge on header icon

### ⏳ Flash Sale Countdown
- Live countdown timer (5h 30min)
- Auto-ticks every second

### 📱 Responsive Design
- Fully responsive: Desktop → Tablet → Mobile
- Sticky header, back-to-top button
- Toast notifications for user feedback

### 📬 Newsletter Signup
- Email validation
- Personalized confirmation message
